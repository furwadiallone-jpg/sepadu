## 🚀 Cara Menggunakan

### **Metode 1: Direct Open (Paling Mudah)**
```
1. Buka folder website
2. Double-click index.html
3. Website langsung terbuka di browser
```

### **Metode 2: Live Server (Rekomendasi)**
```
1. Buka folder website di VS Code
2. Install extension "Live Server"
3. Right-click index.html → Open with Live Server
4. Browser akan buka di http://localhost:5500
```

### **Metode 3: Deploy ke Hosting**
```
1. Upload semua file ke hosting via FTP
2. Pastikan struktur folder tetap sama
3. Update API key Google Maps
4. Website siap live!
```
